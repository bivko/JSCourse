FAQ

Забыл пароль
Если при регистрации вы указывали e-mail адрес, то нужно кликнуть по ссылке «Представиться/зарегистрироваться» в правом верхнем углу, ввести имя участника, ниже нажать кнопку «выслать новый пароль».
Хочу изменить имя участника
Изменить ваше имя участника может только «бюрократ» проекта. Обращайтесь на страницу Википедия:Запросы на переименование учётных записей.
Хочу удалить свою учётную запись
Это невозможно по техническим причинам. Просто перестаньте ею пользоваться.
Насколько родственны проекты Фонда Викимедиа?
Начиная с марта 2008, при регистрации в любом проекте Фонда имя участника автоматически резервируется во всех остальных проектах, подробнее см. meta:Help:Unified login/ru.
